OpenRouter Model Workbook Maintainer V2 expands a workbook-maintainer repo into a daily-refreshable model-intelligence system. It keeps the data pipeline deterministic while turning the results into a visitor-facing guide.

## Why it matters

This project is where price, benchmark coverage, and operator workflow meet. The combination makes it easier to explain not just which model looks strong, but why that conclusion should be trusted.

## What it demonstrates

- OpenRouter plus Artificial Analysis as the core guide cohort
- enrichment layers from Vals and LiveBench without hiding usable models
- deterministic workbook and dataset generation backed by explicit manifests
- a static guide surface that can be refreshed and published on a repeatable operator loop
